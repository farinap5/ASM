# Assembly for 16bist processors

The codes are written to work with windows upcodes.

## Registers
| Register    | 16 | 8  | 8  |
|-------------|----|----|----|
| Accumulator | AX | AH | AL |
| Base        | BX | BH | BL |
| Counter     | CX | CH | CL |
| Data        | DX | DH | DL |
| Souce Index         | SI |
| Target Index        | DI |
| Top Pointer         | BP |
| Base Pointer        | BP |
| Instruction Pointer | IP |
| Code Segment        | CS |
